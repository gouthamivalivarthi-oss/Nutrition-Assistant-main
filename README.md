# Nutrition Assistant (MERN)

## Project Structure
- `frontend/` React + Vite app
- `backend/` Node + Express + MongoDB API

## Backend Setup
1. Open terminal in `backend`
2. Install packages: `npm install`
3. Update `.env` values (`MONGO_URI`, `JWT_SECRET`, `OPENAI_API_KEY`)
4. Run API: `npm run dev`

## Frontend Setup
1. Open terminal in `frontend`
2. Install packages: `npm install`
3. Run app: `npm run dev`

## Features Implemented
- JWT authentication (register/login)
- Protected route for dashboard
- Meal CRUD (create/read/update/delete)
- Nutrition totals (calories, protein, carbs, fats)
- Responsive and attractive UI with nutrition images
- AI diet-plan generation (OpenAI API) with fallback to local smart planner
